# 📊 Credit Risk Assessment System

An end-to-end Machine Learning project that predicts the **probability of loan default**, assigns a **credit score**, and classifies applicants into **risk categories**.

This project simulates a real-world fintech risk evaluation system used by financial institutions to make data-driven lending decisions.

---

## Live Demo
(https://credit-risk-modellingg.streamlit.app/)

---

## Problem Statement

Financial institutions face challenges in identifying high-risk loan applicants.  
Manual evaluation is slow and inconsistent.

This project solves that by:
- Automating credit risk assessment
- Predicting default probability
- Generating a credit score & risk rating

---

## 🔍 Features

- Predicts Default Probability  
- Generates Credit Score (300–900 range)  
- Assigns Risk Rating (Low / Medium / High)  
- Interactive UI using Streamlit  
- Real-time decision insights  

---

## 🤖 Machine Learning Approach

### 1. Data Preprocessing
- Handling missing values  
- Feature scaling  
- Encoding categorical variables  

---

### 2. Model Selection & Optimization

Models evaluated:
- Logistic Regression  
- Decision Tree  
- Random Forest  
- Gradient Boosting  

### Final Model Selection
The final model was selected based on:
- Accuracy  
- Precision & Recall  
- Generalization on unseen data  

Ensemble models like Random Forest / Gradient Boosting performed better because:
- They handle non-linear relationships  
- Reduce overfitting  
- Provide better performance  

---

### 3. Performance Improvement Strategy

- Compared multiple algorithms  
- Tuned hyperparameters  
- Used cross-validation  
- Selected best model with optimal performance
